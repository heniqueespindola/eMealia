# /resume — Retomar Sessão Anterior

## Objectivo

Retoma o trabalho exactamente de onde ficaste na sessão anterior, usando o ficheiro de handoff.

---

## Instruções

Quando invocado com o path do handoff:

1. Lê o ficheiro de handoff fornecido
2. Lê os ficheiros relevantes mencionados (spec, research)
3. Faz um resumo do estado:

```
📋 Retomando sessão de [data]

O que estava a fazer: [descrição]

Progresso:
✅ Concluído: [lista]
⏳ Por fazer: [lista]

Ficheiros activos:
- Spec: [path]
- Research: [path]

Pronto para continuar.
Próximo passo: [acção específica]

Quer que avance directamente?
```

4. Aguarda confirmação antes de continuar a implementação
